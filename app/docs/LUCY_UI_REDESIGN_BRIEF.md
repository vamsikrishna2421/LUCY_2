# Lucy UI Redesign Brief

Goal:
Redesign Lucy app/website to feel like a beautiful passive AI assistant workspace.

Core idea:
Lucy is not just chatbot UI. Lucy is a companion layer that understands life, memories, calendar, tasks, reminders, meetings, productivity, and proactive help.

Visual direction:
- Lumia-inspired workspace tiles
- dark premium background
- soft gradients/glass cards
- minimal text
- big readable sections
- mobile-first
- calm but futuristic
- avoid boring dashboard look

Important screens/areas:
- Home
- Workspace
- Calendar
- Productivity
- Memory
- Notifications / proactive insights
- Assistant detail screen

Notification detail style:
Do not show robotic confidence/memory metadata.
Instead show one human explanation like:
“Lucy brought this up because you were trying to remember it earlier.”

Design task:
First inspect the app. Then propose a design plan. Then implement the first pass screen-by-screen without breaking existing logic.
